# Screenshot upload guide

Læg dine billeder i denne mappe med præcis disse filnavne:

1. `01-dashboard-overview.png`
2. `02-wopr-live-matrix.png`
3. `03-wopr-controls.png`
4. `04-unraid-status-gauges.png`
5. `05-physical-rack-display.jpg`

Tips:
- Brug `.png` til screenshots fra Home Assistant.
- Brug `.jpg` til foto af fysisk hardware.
- Hold billedbredden høj nok til at tekst i dashboard kan læses.
